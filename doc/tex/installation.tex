\chapter{Installation \label{installation}}

\module{lib537} can be installed using either \module{distutils} or
\module{setuptools}. That is, you can either download a tarball, unpack it, and
run:

\begin{verbatim}
$ python setup.py install
\end{verbatim}

Or you can run:

\begin{verbatim}
$ easy_install lib537
\end{verbatim}
